# Dine Ease - Restaurant Website

**Dine Ease** is a modern, responsive frontend web application designed for a premium dining experience. This project features a clean aesthetic with a gold-and-dark theme, providing users with a seamless interface to explore culinary offerings, make reservations, and get in touch.

> **Note:** This project was built for educational purposes only as part of a University course at the University of Sharjah.

---

## 🚀 Features

* **Responsive Landing Page**: A professional home page featuring a hero section with a "Savor the Moment" philosophy and customer reviews.
* **Interactive Menu**: A categorized digital menu (Appetizers, Main Dishes, Desserts, Drinks) with smooth-scrolling navigation and a hover-active category tracker.
* **Reservation System**: A dedicated booking form that allows users to select dates, times, and the number of guests.
* **Dynamic UI Effects**:
* **Fade-in Animations**: Sections reveal themselves as the user scrolls down the page.
* **Glassmorphism & Blurs**: A modern UI design using CSS variables for a "glass" effect and blurred headers.
* **Active Link Highlighting**: The navigation bar automatically highlights the current page.


* **Mobile Optimized**: A fully functional mobile navigation menu triggered by a hamburger icon.

---

## 🛠️ Technologies Used

* **HTML5**: For structured semantic content across all pages (Home, Menu, Reservation, Contact).
* **CSS3**:
* Custom properties (variables) for consistent theming.
* Flexbox and CSS Grid for responsive layouts.
* Advanced animations and transitions.


* **JavaScript (Vanilla)**:
* Intersection Observer API for scroll animations and menu tracking.
* DOM manipulation for the mobile menu and form handling.
* Smooth scrolling functionality.
